# 3D-Print-NAS

Collection of 3D-printable NAS cases.

Requirements:

*   ITX mainboard
    
*   at least 4x 3,5" drives
    
*   Flex, SFX or ATX PSU
    
*   at least somewhat appealing design, not just a slapped together box
