# 3D-Print-NAS

Collection of 3D-printable NAS cases, please [open an issue](https://github.com/FlyingT/3D-Print-NAS/issues/new/choose) or leave a comment on this [Reddit Thread](https://www.reddit.com/r/3DPPC/comments/1ryxkxf/im_trying_to_list_all_3dprintable_nas_models/) if you think this list is missing something!

Requirements:

*   ITX mainboard    
*   at least 4x 3,5" drives
*   Flex, SFX or ATX PSU
*   at least somewhat appealing design, not just a slapped together box
    
The list has been moved to [Google Sheets](https://docs.google.com/spreadsheets/d/1hD1hSavEfwSTJ3hY1r_P37-hqRkqqBagGjUWjJsEd3k/edit?usp=sharing) for easier editing:
